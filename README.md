# WeatherApp
  Simple weather app built with Next.js and Tailwind CSS
  Weather information is being supplied by OpenWeatherMap.org
  


## Technology
  1. Next.js
  2. OpenWeatherMap
  3. Axios
  4. Tailwind CSS
